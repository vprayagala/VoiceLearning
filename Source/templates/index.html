This Page is Just to Demonstarte Sentiment Analysis

Upload an audio file (only WAV file format audio) or enter text to get the sentiment score. Audio will be converted to text using cloud speechAPI and then bi-directional LSTM model is used to determine the sentiment score. Bi-directional LSTM is trained using the IMDB data source available over web


Select an option:

Derive Sentiment From Text
